# Job 77 – Northframe Movie System (Focused Blueprint)

## Purpose
Convert PDFs, documents, scripts, and ideas into professional AI-generated videos (shorts, mid-form, and feature-length movies).

## Core Flow (Simplified Straight Pipeline)
Upload (PDF / Document / Idea)
↓
PDF Analysis + Script Generation
↓
Storyboard Creation
↓
Cinematic Planning (Camera + Motion + Laws)
↓
Scene Building
↓
Rendering
↓
Governor Validation (Quality + Continuity)
↓
Export + Metadata
↓
Publish (Marketplace / Direct Delivery)
## Key Principles
- Straight pipeline (no unnecessary complexity)
- PDF is the source of truth
- Every movie follows cinematic laws
- Governor must approve before delivery
- Designed to work with NeverX007 + Grok + Gemini

## Current Tech
- FastAPI backend (already running)
- SQLite for projects and logs
- FFmpeg for basic rendering
- Future: Gemini for planning/scripting, Grok for orchestration

## What We’re Keeping from Old Blueprint
- PDF → Movie pipeline
- Storyboard concept
- Cinematic laws / continuity rules
- Governor validation step
- Clear modular workflows

## What We’re Dropping
- Everything outside Job 77
- Pocketbase
- Old job numbers (73-76)
- Unrelated courses and side projects
